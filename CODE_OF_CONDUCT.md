# Code of Conduct

## Summary
This project follows a professional, respectful collaboration standard focused on technical rigor and constructive review.

## Standards
- Be respectful and specific in feedback.
- Challenge ideas, not people.
- Prefer evidence from incidents, benchmarks, or formal reasoning.
- Assume good intent and clarify before escalating.

## Enforcement
Report unacceptable behavior to repository maintainers through private channels.
