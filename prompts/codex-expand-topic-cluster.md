# Prompt: Expand Topic Cluster

You are extending this repository with a new concept cluster.

## Inputs
- Cluster name
- 5-10 target pages
- Existing related pages

## Tasks
1. Propose cluster scope and boundaries.
2. Generate high-quality pages using repository template style.
3. Add cross-links and `Related concepts` sections.
4. Update `schemas/knowledge.json`, `schemas/content-index.json`, and `schemas/topic-graph.json`.
5. Suggest diagram updates if relationships changed.

## Output format
- Files created/updated
- Rationale for taxonomy changes
- Validation checklist
