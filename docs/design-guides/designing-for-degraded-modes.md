# Designing For Degraded Modes

**Summary:** Practical design guidance for designing for degraded modes.

## Decision framework
Define objectives, protected paths, capacity boundaries, and rollback criteria.

## Implementation guidance
Use explicit policies, reason codes, and measurable thresholds.

## Validation
Run load tests and failure drills that include retries and dependency faults.
